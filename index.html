Currently in triathlons, it is only possible to see the athletes position after they have finished a leg, such as the Swim Bike or Run.

This can be frustrating to some spectators when it is a large race and they cannot see their loved ones or they cannot be at the race in person but would like to be able to see how the athlete is doing.

This system is design to give almost up to the minute information as to the current position of the athlete using GPS tracking pods that are attached to the person.

These pods store the GPS data and within the route there are retrieval stations that automatically retrive the data as the athlete passes. This data is then uploaded to a website that spectators can see the positioning as well as allowing the athlete to download the data after the event if they with to analyse it is more detail.

General Questions

How does the system work?
Each athlete has a tracking unit attached, that maps their current position. At set intervals there are readers that retrieve the information from the unit as it passes and stores it for the athlete. This will then be viewable real time by supporters wanting to track the athlete as well as to the athlete to download at a later stage.
Is the unit heavy?
No, it has been designed as light as possible not to impede the athlete and will be similar to current tracking systems.
